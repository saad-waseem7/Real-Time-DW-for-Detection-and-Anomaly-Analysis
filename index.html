🔍 Fraud Detection Data Warehouse

Detect fraud, analyze patterns, and prevent financial losses

📌Overview

A system that detects fraudulent financial transactions, organizes data, and shows insights through visual dashboards. Perfect for banks and payment companies who want to catch fraud in real-time.

🎯What It Does

  • Identifies suspicious transactions automatically
  • Organizes messy data into clear structure
  • Analyzes fraud patterns by region, time, and device
  • Creates interactive dashboards for business users

🛠️Technology Stack

PostgreSQL pgAdmin4 Power BI SQL

📁Project Folders

📂 D1/ Database setup and table structure
📂 D2/ Analysis queries and reports
📂 D3/ Database speed optimization
📂 D4/ Power BI dashboards and visualizations
📂 csv files/ Sample financial transaction data

🚀Quick Start

  1. Install PostgreSQL and pgAdmin4
  2. Load CSV files into database
  3. Run SQL scripts from D1 folder to create tables
  4. Check results in Power BI dashboard (D4)

⭐Key Features

Fraud Detection

Automatically identifies suspicious transactions and patterns

Data Organization

Converts messy data into organized warehouse structure

Real-time Analysis

Analyzes fraud by region, time, device, and user behavior

Interactive Dashboards

Visual reports for easy business decision making
